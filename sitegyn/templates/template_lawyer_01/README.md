# template_lawyer_01

Professional multi-page legal services template for Sitegyn.

## Features
- Multi-page support (Home, About, Services, Contact)
- Full Editor AI actions support
- JSON-driven rendering (schema + mapping)
- Realtime preview updates
- Clean, production-ready HTML

## Pages
- Home
- About
- Services
- Contact

## Sections
- Hero
- About + Why Us
- Services
- Offers
- Contact

## Notes
This template is fully driven by `content_json` and does not rely on hardcoded pages.
